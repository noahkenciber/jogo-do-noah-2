# 🚀 Como Publicar seu Jogo no GitHub

Siga estes passos simples para colocar o **Jogo do Noah 2** online!

## 1. GitHub Desktop (Commit)

1.  Abra o **GitHub Desktop**.
2.  Clique em **Add** > **Add existing repository** e selecione a pasta: `C:\Users\26012260\Documents\GitHub\jogo do noah 2`.
3.  No canto inferior esquerdo, escreva uma mensagem de commit, como: `Primeira versão do Jogo do Noah 2`.
4.  Clique em **Commit to main**.
5.  Clique em **Publish repository** (ou **Push origin**) no topo.

## 2. GitHub Pages (Deploy)

1.  Vá até o site do GitHub no seu navegador e abra o repositório `jogo-do-noah-2`.
2.  Clique na aba **Settings** (Configurações) no topo.
3.  No menu lateral esquerdo, clique em **Pages**.
4.  Em **Build and deployment** > **Branch**, verifique se está selecionado `main` e `/(root)`.
5.  Clique em **Save**.
6.  Aguarde cerca de 2 minutos. Um link aparecerá no topo da página (ex: `https://seu-usuario.github.io/jogo-do-noah-2/`).

## 3. Como Jogar

- **No Computador**: Basta abrir o arquivo `index.html` arrastando-o para o Chrome ou Edge.
- **No Celular**: Após o deploy no GitHub Pages, você poderá abrir o link no seu celular e o jogo funcionará com as faixas de toque (em breve adicionarei suporte a swipe!).
